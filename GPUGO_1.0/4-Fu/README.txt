# Project Name

GPU-Based Global Optimization (GPUGO)

## File Descriptions

The folder “4-Fu” contains five Python scripts below: 

- IATrigonometric.py: The Python script “IATrigonometric.py” contains the 
interval arithmetic operations for evaluating the objective function 
(Trigonometric Function). The functions in this script are called by the main 
runnable script.

- trigonometric50.py: The Python script “trigonometric50.py” is the main 
script that finds the global minimum of the Trigonometric Function using the 
GPU-based global optimization method. The objective function has 50 variables 
in this script.

- trigonometric100.py: The Python script “trigonometric100.py” is the main 
script that finds the global minimum of the Trigonometric Function using the 
GPU-based global optimization method. The objective function has 100 variables 
in this script.

- trigonometric500.py: The Python script “trigonometric500.py” is the main 
script that finds the global minimum of the Trigonometric Function using the 
GPU-based global optimization method. The objective function has 500 variables 
in this script.

- trigonometric1000.py: The Python script “trigonometric1000.py” is the main 
script that finds the global minimum of the Trigonometric Function using the 
GPU-based global optimization method. The objective function has 1000 variables 
in this script.

## Execution Instructions
The execution of the Python scripts in the folder “4-Fu” could 
follow the three steps below.

1. Set up the environment based on the Environment section.
2. Ensure the “IATrigonometric.py” script is in the same directory as the main 
script.
3. Run the main script “trigonometricXXXX.py” in Python.
