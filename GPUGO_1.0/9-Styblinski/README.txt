# Project Name

GPU-Based Global Optimization (GPUGO)

## File Descriptions

The folder “9-Styblinski” contains five Python scripts below: 

- IACorrupted.py: The Python script “IACorrupted.py” contains the interval 
arithmetic operations for evaluating the objective function (Multimodal 
Corrupted Convex Quadratic Function). The functions in this script are called 
by the main runnable script.

- corrupted50.py: The Python script “corrupted50.py” is the main script that 
finds the global minimum of the Multimodal Corrupted Convex Quadratic Function 
using the GPU-based global optimization method. The objective function has 50 
variables in this script.

- corrupted100.py: The Python script “corrupted100.py” is the main script that 
finds the global minimum of the Multimodal Corrupted Convex Quadratic Function 
using the GPU-based global optimization method. The objective function has 100 
variables in this script.

- corrupted500.py: The Python script “corrupted500.py” is the main script that 
finds the global minimum of the Multimodal Corrupted Convex Quadratic Function 
using the GPU-based global optimization method. The objective function has 500 
variables in this script.

- corrupted1000.py: The Python script “corrupted1000.py” is the main script that 
finds the global minimum of the Multimodal Corrupted Convex Quadratic Function 
using the GPU-based global optimization method. The objective function has 1000 
variables in this script.

## Execution Instructions
The execution of the Python scripts in the folder “9-Styblinski” could follow 
the three steps below.

1. Set up the environment based on the Environment section.
2. Ensure the “IACorrupted.py” script is in the same directory as the main 
script.
3. Run the main script “corruptedXXXX.py” in Python.
