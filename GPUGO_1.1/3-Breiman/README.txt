# Project Name

GPU-Based Global Optimization (GPUGO)

## File Descriptions

The folder “3-Breiman” contains five Python scripts below: 

- IACosine.py: The Python script “IACosine.py” contains the interval arithmetic 
operations for evaluating the objective function (Cosine Mixture Function). The 
functions in this script are called by the main runnable script.

- cosine50.py: The Python script “cosine50.py” is the main script that 
finds the global minimum of the Cosine Mixture Function using the GPU-based 
global optimization method. The objective function has 50 variables in this 
script.

- cosine100.py: The Python script “cosine100.py” is the main script that 
finds the global minimum of the Cosine Mixture Function using the GPU-based 
global optimization method. The objective function has 100 variables in this 
script.

- cosine500.py: The Python script “cosine500.py” is the main script that 
finds the global minimum of the Cosine Mixture Function using the GPU-based 
global optimization method. The objective function has 500 variables in this 
script.

- cosine1000.py: The Python script “cosine1000.py” is the main script that 
finds the global minimum of the Cosine Mixture Function using the GPU-based 
global optimization method. The objective function has 1000 variables in this 
script.

## Execution Instructions
The execution of the Python scripts in the folder “3-Breiman” could 
follow the three steps below.

1. Set up the environment based on the Environment section.
2. Ensure the “IACosine.py” script is in the same directory as the main script.
3. Run the main script “cosineXXXX.py” in Python.
