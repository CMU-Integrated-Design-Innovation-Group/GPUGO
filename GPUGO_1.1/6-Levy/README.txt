# Project Name

GPU-Based Global Optimization (GPUGO)

## File Descriptions

The folder “6-Levy” contains eight Python scripts below: 

- IALevy.py: The Python script “IALevy.py” contains the interval arithmetic 
operations for evaluating the objective function (Levy Function). The functions 
in this script are called by the main runnable script.

- levy50.py: The Python script “levy50.py” is the main script that finds the 
global minimum of the Levy Function using the GPU-based global optimization 
method. The objective function has 50 variables in this script.

- levy100.py: The Python script “levy100.py” is the main script that finds the 
global minimum of the Levy Function using the GPU-based global optimization 
method. The objective function has 100 variables in this script.

- levy500.py: The Python script “levy500.py” is the main script that finds the 
global minimum of the Levy Function using the GPU-based global optimization 
method. The objective function has 500 variables in this script.

- levy1000.py: The Python script “levy1000.py” is the main script that finds 
the global minimum of the Levy Function using the GPU-based global optimization 
method. The objective function has 1000 variables in this script.

- levy2000.py: The Python script “levy2000.py” is the main script that finds 
the global minimum of the Levy Function using the GPU-based global optimization 
method. The objective function has 2000 variables in this script.

- levy5000.py: The Python script “levy5000.py” is the main script that finds 
the global minimum of the Levy Function using the GPU-based global optimization

- levy10000.py: The Python script “levy10000.py” is the main script that finds 
the global minimum of the Levy Function using the GPU-based global optimization 
method. The objective function has 10000 variables in this script.

## Execution Instructions
The execution of the Python scripts in the folder “6-Levy” could follow the 
three steps below.

1. Set up the environment based on the Environment section.
2. Ensure the “IALevy.py” script is in the same directory as the main script.
3. Run the main script in the format “levyXXXX.py” in Python.
