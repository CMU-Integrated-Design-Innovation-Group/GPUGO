# Project Name

GPU-Based Global Optimization (GPUGO)

## File Descriptions

The folder “10-Zabinsky” contains five Python scripts below: 

- IASinusoidal.py: The Python script “IASinusoidal.py” contains the interval 
arithmetic operations for evaluating the objective function (Sinusoidal 
Function). The functions in this script are called by the main runnable script.

- sinusoidal50.py: The Python script “sinusoidal50.py” is the main script 
that finds the global minimum of the Sinusoidal Function using the GPU-based 
global optimization method. The objective function has 50 variables in this 
script.

- sinusoidal100.py: The Python script “sinusoidal100.py” is the main script 
that finds the global minimum of the Sinusoidal Function using the GPU-based 
global optimization method. The objective function has 100 variables in this 
script.

- sinusoidal500.py: The Python script “sinusoidal500.py” is the main script 
that finds the global minimum of the Sinusoidal Function using the GPU-based 
global optimization method. The objective function has 500 variables in this 
script.

- sinusoidal1000.py: The Python script “sinusoidal1000.py” is the main script 
that finds the global minimum of the Sinusoidal Function using the GPU-based 
global optimization method. The objective function has 1000 variables in this 
script.

## Execution Instructions
The execution of the Python scripts in the folder “10-Zabinsky” could follow 
the three steps below.

1. Set up the environment based on the Environment section.
2. Ensure the “IASinusoidal.py” script is in the same directory as the main 
script.
3. Run the main script “sinusoidalXXXX.py” in Python.
